# Python 3 – Mundo 2

Exercícios focados em lógica de programação, estruturas condicionais e laços de repetição.

## Conteúdos praticados
- if / elif / else
- while e while True
- Contadores e acumuladores
- Validação de entrada de dados
- Resolução de problemas práticos

Alguns exercícios simulam situações reais, como caixa eletrônico, cadastro de pessoas e sistema de compras.
